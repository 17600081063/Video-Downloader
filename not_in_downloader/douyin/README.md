# Douyin App - Video Downloader

# Usage:
### Step1-download this repository:
Clone or download.
### Step2-Install some dependencies:
"pip install -r requirements.txt" or  
"py -3 -m pip install -r requirements.txt"  
### Step3-Run douyin.py:
"python douyin.py" or "py -3 douyin.py"
